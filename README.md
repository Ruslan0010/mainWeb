# Atlantic AI Solutions

Лендинг-страница AI-агентства из Галифакса (Новая Шотландия). Одностраничный сайт с героем, кейсами, услугами и формой обратной связи.

- **GitHub Pages:** https://ruslan0010.github.io/mainWeb/
- **Vercel (production):** https://mainweb-m8nt65lg9-ruslans-projects-fbbc69de.vercel.app
- **Репозиторий:** https://github.com/Ruslan0010/mainWeb

## 📖 О проекте

Atlantic AI Solutions помогает бизнесу в Галифаксе автоматизировать рутину, внедрять AI-ассистентов и масштабироваться без найма. Сайт презентует услуги агентства:

- Sales Automation
- AI Chatbots
- Process Automation
- Custom Dashboards
- System Integrations

## 🛠 Стек

- Чистый **HTML/CSS/JS** — без сборки, одним файлом `index.html`
- **GSAP + ScrollTrigger** — анимации при скролле
- **Lenis** — плавный скролл
- **Fontshare** — шрифты Cabinet Grotesk и Satoshi

## 🚀 Локальный запуск

```bash
# любой статический сервер, например:
python3 -m http.server 8000
# → http://localhost:8000
```

Или просто откройте `index.html` в браузере.

## 📦 Деплой

- **GitHub Pages** — автоматически из ветки `main` (корень репозитория).
- **Vercel** — `vercel deploy --prod` из корня проекта.
